# bof-ai-dashboard PLAN

## 概要
AI活用プロジェクトのダッシュボード。2つのバージョンを管理:
- **index.html（ポートフォリオ版）**: 全社公開用・外部向けポートフォリオ
- **internal.html（社内用）**: 社内メンバー向けプロジェクト管理ダッシュボード

## ファイル構成
```
bof-ai-dashboard/
├── index.html          ← ポートフォリオ版（外部向け）
├── internal.html        ← 社内用ダッシュボード
├── css/
│   ├── style.css        ← ポートフォリオ用CSS
│   └── internal.css     ← 社内用CSS
├── js/
│   ├── data.js          ← 共有データ（両方で使用）
│   └── main.js          ← ポートフォリオ用JS
└── PLAN.md              ← このファイル
```

| 構想中（planned） | アイデア段階 | #A0A0A0（グレー） |
